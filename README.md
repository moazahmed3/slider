# 🖼️ Image Slider (Lightbox Gallery)

A simple and responsive Image Slider (Lightbox Gallery) built with **HTML, CSS, Bootstrap, and JavaScript**.

Users can click any image to open it in a lightbox, navigate through images using buttons or keyboard shortcuts, and close the lightbox in multiple ways.

---

## 🚀 Features

- 📸 Open image in a lightbox by clicking on it
- ⬅️ Previous image navigation
- ➡️ Next image navigation
- 🔄 Infinite image looping
- ❌ Close using the close button
- ⌨️ Keyboard support
  - `Esc` → Close lightbox
  - `←` Left Arrow → Previous image
  - `→` Right Arrow → Next image
- 🖱️ Close by clicking outside the image
- 📱 Responsive design

## ⚙️ How It Works

1. Click on any gallery image.
2. The image opens inside the lightbox.
3. Navigate using:
   - Previous button
   - Next button
   - Keyboard arrows
4. Close the lightbox by:
   - Clicking the close icon
   - Pressing `Esc`
   - Clicking outside the image

---
